## Documented Limitations

```text
- No post-creation manual route vertex editing.
- No component rotation or mirror controls.
- No drag-from-palette placement.
- No live ERC while editing.
- No net label placement objects, buses, hierarchical sheets, or symbol library editor.
- No multi-select, copy/paste, grouping, or bulk operations.
- Browser smoke can inspect the Vite-rendered UI; native Tauri filesystem/dialog behavior still needs manual desktop smoke.
- Native Tauri manual smoke is still recommended after the post-video repair.
- Full KiCad/Altium/EasyEDA parity remains out of scope.
- No RF, Smith chart, S-parameter hardening, or new analysis workflow was added in v3.6-pre-ui-polish.
```

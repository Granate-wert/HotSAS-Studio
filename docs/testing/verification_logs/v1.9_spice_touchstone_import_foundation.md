## 6. Known limitations / deferred work

1. **SPICE parser is foundation only**: parses `.model` and `.subckt` with parameters. Does not handle `.include`, `.lib`, `.param`, `.func`, `.control` — these generate warnings.
2. **Touchstone plotting**: parsed data is stored but not visualized. Future stage will add S-parameter charts.
3. **Pin mapping UI**: backend has `validate_spice_pin_mapping` and `attach_imported_model_to_component`, but the frontend Import Models screen does not yet have an interactive pin mapper. It shows model details only.
4. **File import**: currently text-paste only. File picker integration deferred.
5. **Vendor library manager**: deferred to future stage.

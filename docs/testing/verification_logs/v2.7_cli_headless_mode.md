## Notes

- CLI crate (`hotsas_cli`) delegates 100% to existing `HotSasApi` facade methods. No business logic duplication.
- Exit codes enforced: 0 success, 1 internal/IO, 2 validation/input, 3 usage, 4 unsupported.
- `csv-summary` export delegates to `AdvancedReportService::render_report_csv_summary()` instead of manual CSV construction.
- `FormulaPackFileLoader::load_builtin_packs()` used for headless initialization; falls back to built-in RC formula if packs directory is unreachable.
- `CircuitProjectPackageStorage` requires `.circuit` extension on package directory name (reflected in integration tests).
- `run_simulation` supports `"ac_sweep"` and `"transient"` analysis kinds.
- All 361 Rust tests pass including 10 new CLI integration tests.

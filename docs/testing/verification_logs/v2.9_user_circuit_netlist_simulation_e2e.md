## What Was Verified

1. `SimulationWorkflowService` orchestrates preflight → netlist → engine dispatch → result → cache.
2. `UserCircuitSpiceNetlistExporter` generates SPICE from arbitrary user circuits (R, C, L, V, D).
3. Ground nets renamed to `0`; `ground_reference` detected.
4. Netlist reads `overridden_parameters` set via QuickParameterEditor.
5. Profile list returns 4 profiles (OP, AC, Transient, Auto).
6. Probe suggestion scans nets.
7. Preflight validates components, nets, probes, ground, netlist generation.
8. Mock engine runs all three analysis types.
9. Auto fallback produces explicit warning when ngspice unavailable.
10. Result contains measurements + series + status + engine.
11. Last-run cache is session-local only.
12. Report section generated with blocks + warnings.
13. API facade exposes 7 methods.
14. Tauri commands wired with permissions.
15. CLI `user-circuit-simulate` works on saved user project.
16. No frontend SPICE generation.
17. No frontend ngspice execution.
18. React remains view adapter only.

## Manual Smoke Test (optional)

- [ ] Open Schematic Editor.
- [ ] Build RC circuit (Vsource → R → C → GND).
- [ ] Set R=10k, C=100n via Quick Parameter Editor.
- [ ] Open Simulation tab in bottom panel.
- [ ] Select "AC Sweep (Mock)" profile.
- [ ] Run preflight — validate passes.
- [ ] Click Run Simulation.
- [ ] Verify status = Succeeded, engine = mock.
- [ ] Verify measurements table shows values.
- [ ] Verify graph series renders.
